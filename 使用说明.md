# 政务服务平台登录页面使用说明

## ✅ 已完成功能

1. **登录页面设计**
   - 左右分栏布局
   - 左侧：蓝色背景，展示平台特色和功能介绍
   - 右侧：白色背景，登录表单
   - 响应式设计，支持不同屏幕尺寸

2. **功能特性**
   - 用户名和密码输入
   - 密码显示/隐藏切换
   - 记住我功能
   - 忘记密码链接
   - 注册链接
   - 表单验证

## 🚀 启动应用

### 方式一：使用启动脚本
```powershell
.\start.ps1
```

### 方式二：手动启动
```powershell
.\venv\Scripts\Activate.ps1
python run.py
```

## 📱 访问页面

启动应用后，在浏览器中访问：

- **登录页面**: http://localhost:5000/login
- **首页**: http://localhost:5000 （自动重定向到登录页）

## 🎨 页面说明

### 左侧面板
- 平台Logo和名称
- 主标题："便捷高效的政务服务体验"
- 平台介绍
- 三大特性：
  - 安全可靠
  - 高效便捷
  - 7×24小时服务

### 右侧面板
- 登录表单
- 用户名输入框（带图标）
- 密码输入框（带显示/隐藏切换）
- 记住我选项
- 登录按钮
- 注册链接

## 🔧 技术栈

- **前端**: HTML5 + CSS3 + JavaScript + Layui

## 📞 问题排查

如果页面无法打开，请检查：

1. 应用是否正常启动（查看终端输出）
2. 端口5000是否被占用
3. 浏览器控制台是否有错误信息
4. 静态文件路径是否正确
